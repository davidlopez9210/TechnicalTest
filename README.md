# TechnicalTest

## Description

> Technical test developed with Selenium Framework by David López
> 
> This proyect is based in Selenium framework with .NET Core 3.1 and NUnit testing framework.
> 
>It also uses Google Chrome and Mozilla Firefox WebDrivers for running automated tests on both browsers with one common code.


## Usage

> To run tests, open the Visual Studio solution and go to the menu Test => Test Explorer.
> In that view you can run or debug the automated tests by right clicking each scenario.
